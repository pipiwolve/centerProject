# 边油开裂 vs 手柄本体损伤：什么时候只是 edge painting，什么时候要换提手

## 资料定位

这是一张高价值判别卡，专门解决“看起来像边油裂，其实已经伤到提手本体或连接位”的问题。适合肩带、提手、包耳、puller、边缘收口等场景。

## 检索标签

- 症状：边油开裂、包边裂、提手开裂、手柄掉皮、puller 坏了、边缘掉层
- 典型问法：`边油裂了怎么修`、`提手开裂要换吗`、`只补边油还是换手柄`
- 风险等级：高

## 适用判断

适用于以下情况：

- 破损集中在提手、肩带、包耳、puller、边缘收口。
- 用户无法判断是 cosmetic edge issue 还是结构性损伤。

## 判别要点

1. 如果裂纹只在边缘涂层、表面圆边仍完整，通常更接近 edge-paint 类问题。
2. 如果已经牵涉提手本体、连接位、拉力点或层间开口，就不应再按单纯补边油处理。
3. 这个区分不是空想：LOEWE 官方维修菜单把 `edge painting` 和 `handle replacement` 列为不同服务；Longchamp 也把 handles/straps repair 单列出来。

以下判断为基于官方维修项目划分的工程性推断：

- `edge painting` 更偏向表层边缘修整；
- `handle replacement`、`strap repair`、`re-stitching`、`puller replacement` 更偏向部件或结构层级处理。

## 处理目标

- 先判断问题层级。
- 避免把结构问题误判成表层 cosmetic 问题。

## 推荐工具

- 干布
- 照片记录

## 操作步骤

1. 先看位置：是在边缘表层，还是已经进入提手根部、孔位、五金连接位。
2. 先看受力：只要该部位承担拉力，就按更高风险处理。
3. 先停用重载：不要让已经裂开的提手继续受重。
4. 记录裂口形态：是表层边缘脱层，还是层间开口、断裂、露出基材。
5. 根据层级分流：表层边缘问题可咨询 edge painting；本体、连接位和拉力点问题应直接咨询 handle/strap replacement 或 re-stitching。

## 不建议的做法

- 不要继续靠单纯补边油遮住已经开裂的真皮层。
- 不要对承重位做家庭胶补。
- 不要等到提手断裂后再进入维修流程。

## 直接送修 / 升级处理

- 提手、肩带、包耳、puller 已经影响使用安全。
- 裂纹已从边缘进入本体。
- 连接位、孔位、铆钉位、缝线同时异常。
- 用户希望维持原件结构和品牌一致性。
