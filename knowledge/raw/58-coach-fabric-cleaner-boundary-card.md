# Coach Fabric Cleaner 边界卡：什么能用，什么绝对不能用

## 资料定位

这是一张专门约束 `Coach Fabric Cleaner` 使用场景的规则卡。非常适合生产端，因为用户看到“官方 fabric cleaner”后，很容易误解成一切织物拼皮包都能直接上。

## 可用范围

- Coated Canvas
- fabric / jacquard

## 明确不可用范围

- leather bags / trims
- suede or treated suede bags / trims
- haircalf
- metallic finish fabrics
- twill
- straw
- nylon
- denim
- satin / polysatin
- wool
- laminated finishes

## 路由重点

1. 只要包体有 leather trim，就不能把 Fabric Cleaner 一路擦过去。
2. `nylon` 被官方明确排除，不能因为“它也是布”就套用。
3. `Coated Canvas` 可用，并不等于所有 canvas-like 材质都可用。

## 操作逻辑

1. 先识别材质和 trim。
2. 只在确认是 Coated Canvas 或 fabric / jacquard 时进入 Fabric Cleaner 路线。
3. 只用 soft white cloth，小范围 gentle circular motions。
4. 后续用 clean white dampened cloth 擦掉 excess，并 blot dry。

## 不建议的做法

- 不要对 leather trim 使用 Fabric Cleaner。
- 不要把它拿去清 suede、haircalf 或 nylon。
- 不要因为是“官方产品”就跳过材质确认。
