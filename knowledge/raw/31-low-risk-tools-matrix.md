# 低风险工具矩阵：先用什么、哪些材质能用、哪些情况不该用

## 资料定位

这份矩阵不是“产品推荐榜”，而是 RAG 用的低风险工具路由卡。目标是把用户问题先分到正确的工具层级，再决定是否继续进入清洁、护理或送修。

## 使用总原则

1. 先选物理工具，再决定是否需要化学品。
2. 能干式处理时，不先上湿式方案。
3. 能局部测试时，不先整面处理。
4. 工具和材质要一一对应，不做跨材质套用。

## 工具矩阵

### 软质无绒布 / 浅色吸水布

- 适用：光面皮、植鞣革的临时吸水、漆皮表层轻拭、涂层帆布、受潮止损
- 不适用：翻毛皮的重度来回摩擦
- 适合任务：吸附水分、带走浮灰、微湿点擦
- 风险提醒：深色布可能掉色，起毛布会把纤维留在皮面上

### 软毛刷 / 马毛刷

- 适用：植鞣革表面浮灰、光面皮接缝灰尘、五金附近干式除尘
- 不适用：小羊皮高压刷拭、漆皮反复刷洗
- 适合任务：先清掉浮灰，再决定要不要进入下一步
- 风险提醒：只做轻刷，不把污物压进皮面

### 翻毛皮专用刷

- 适用：suede、nubuck、roughout
- 不适用：漆皮、光面皮、植鞣革本色皮
- 适合任务：梳理绒毛、恢复方向、处理轻度白印
- 风险提醒：必须在完全干燥状态下使用

### Suede Eraser / 翻毛皮橡皮

- 适用：翻毛皮轻度表面痕、白印、轻度摩擦痕
- 不适用：油斑、湿态污渍、已经结块发硬的区域
- 适合任务：先做干式小范围试探
- 风险提醒：压力过大容易造成局部发白

### 雪梨纸 / 无酸纸 / 轻柔填充物

- 适用：包型维持、受潮后轻撑、长期收纳
- 不适用：塞得过满的“硬顶型”支撑
- 适合任务：防塌、防折、防链条压痕
- 风险提醒：不使用油墨重、偏硬或边角锋利的纸张

### 防尘袋 / 透气布袋

- 适用：大多数包袋的长期存放
- 不适用：湿包尚未干透时的封存
- 适合任务：避灰、减摩擦、隔离染色源
- 风险提醒：不透气塑料袋不作为长期收纳方案

### 柔软包裹物

- 适用：链条、肩带、五金与皮面的隔离
- 不适用：会掉毛、会掉色、边缘粗糙的材料
- 适合任务：防压痕、防串色、防五金长期压迫

## 快速路由

- 用户说“只是有灰和轻印子”：先路由到干布、软毛刷。
- 用户说“翻毛皮局部发暗”：先路由到翻毛皮专用刷和橡皮，再判断是否需要 suede cleaner。
- 用户说“刚淋到水”：先路由到浅色吸水布和轻撑纸张，不进入上油逻辑。
- 用户说“收纳变形”：先路由到填充物、防尘袋和链条隔离，不先路由到护理油。
