# 小皮件翻盖翘边、口沿卷曲与边油起翘分流卡：变形还是 edge finish 失效

## 资料定位

这是一张针对 `wallet flap / card holder edge / glazing lift / edge curl` 的细颗粒场景卡，适合处理 `翻盖翘起来了`、`边油起皮了`、`卡包口沿卷了` 这类问法。

## 检索标签

- 品类：wallet、card holder、small leather goods、coin purse、SLG
- 部位：flap、edge、glazing、口沿、翻盖边、折边
- 症状：翘边、卷边、边油起翘、边缘发黏、开胶、翻盖不贴合
- 风险等级：中高

## 适用判断

适用于以下情况：

- 问题发生在翻盖、卡槽口、包口折边等高频弯折部位。
- 用户不确定这是储存形变，还是已经进入 edge finish / adhesive failure。

## 判别要点

1. CHANEL 官方对 small leather goods 强调不要 overfill，并避免 humidity、light 与会改变 finish 的 care products。
2. Goyard 官方明确警告：过载和不当用途可能损伤材料并 alter edge finishes。
3. LOEWE ReCraft 官方维修菜单单列 edge painting 与 re-stitching，说明边缘问题达到一定程度后应进入维修路径。

## 路由规则

### 可观察层

- 只是轻微翘起，边油仍连续，没有裂口、发黏或脱层。
- 翻盖仍能正常贴合，开合手感基本稳定。
- 问题更像长期压放、装太满或暂时受潮后的形变。

### 维修层

- 边油起翘、裂开、发黏、泛白或开始粘手。
- 翻盖边缘已经分层，或折边开始张口。
- 翘边同时影响开合贴合度、卡片保持力或整体外观对称性。

## 推荐工具

- 干布
- 空载静置
- 拍照记录

## 操作步骤

1. 先减载：把内部卡片和票据恢复到正常容量。
2. 干燥静置：避免继续受压、受热或潮湿环境。
3. 观察边缘层级：区分是单纯卷曲，还是已经连带边油、胶层、缝线变化。
4. 进入维修：若边油已起翘或翻盖贴合受影响，转 edge painting / re-stitching 路由。

## 不建议的做法

- 不要用胶水、热风、熨压或重物夹压强行压平。
- 不要用酒精、溶剂或强清洁剂擦边油。
- 不要把边油问题继续当成单纯储存变形拖延。

## 直接送修 / 升级处理

- 边油已经裂、翘、黏或脱层。
- 翻盖无法平整贴合，影响开合和外观。
- 边缘问题已扩展到折边、缝线或卡槽口。
- 用户追求恢复原有边缘厚度、线条和平直度。
