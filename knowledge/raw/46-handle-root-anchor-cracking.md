# 手柄根部、肩带挂点与受力连接位开裂：高优先级维修分流卡

## 资料定位

这是一张针对 `handle root / strap anchor / attachment point / 提手根部 / 肩带挂点` 的结构风险卡。适合看起来“只是边缘裂一点”，但位置处于承重点的问题。

## 检索标签

- 部位：手柄根部、肩带挂点、连接位、包耳、attachment point、anchor point
- 症状：开裂、变形、扯开、受力处松了、连接位裂开
- 典型问法：`提手根部裂了怎么办`、`肩带连接处开裂`、`挂点松了`
- 风险等级：高

## 适用判断

适用于以下情况：

- 裂纹发生在手柄根部、肩带孔位、包耳、挂点、铆钉位等承重点。
- 用户仍在正常背负或打算继续装重物。

## 判别要点

1. 这类位置的破损不是普通表层问题，而是承重安全问题。
2. Longchamp 官方将 handles/straps damage 直接路由到 Repair Center；LOEWE ReCraft 把 handle replacement 单列；Coach Repairs 也将 stitching / hardware replacement 列为维修服务。
3. 因此，只要裂纹出现在连接位，就不该按单纯补边油、补色或表面清洁处理。

## 处理目标

- 立即降低受力。
- 阻止小裂纹扩成断裂、脱落或硬件牵连损伤。

## 推荐工具

- 干布
- 拍照记录
- 防尘袋

## 操作步骤

1. 立即减重或停用：不要继续装电脑、书、本体重物。
2. 检查受力路径：看是否同时牵连缝线、铆钉、边缘收口或相邻五金。
3. 记录裂口走向：判断是边缘表层、层间开口，还是已经进入本体。
4. 维持自然形态：收纳时避免让该连接位继续扭曲或悬吊。

## 不建议的做法

- 不要用家用胶水、热熔胶、皮贴或针线自行做承重修补。
- 不要因为“还能拎”就继续照常使用。
- 不要只补边缘涂层，忽视内部结构层已经开裂。

## 直接送修 / 升级处理

- 裂纹位于承重点。
- 伴随缝线、五金、孔位、铆钉或包耳一起异常。
- 裂纹在继续扩大或已影响使用信心。
- 用户希望保持原有提手/肩带长度、比例和品牌工艺。
