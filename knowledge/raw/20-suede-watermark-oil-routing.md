# 翻毛皮 / Nubuck：水印、油斑与局部发黑的分流判断

## 资料定位

这是一张针对 `suede / nubuck / roughout / 翻毛皮 / 磨砂皮 / 反绒皮` 的判别型场景卡，重点不是“一招洗净”，而是先把不同污渍路线分开。

## 检索标签

- 材质：翻毛皮、反绒皮、suede、nubuck、roughout、磨砂皮
- 症状：水印、油斑、发黑、局部变暗、绒毛倒伏、结块
- 典型问法：`翻毛皮发黑怎么办`、`suede water stain`、`nubuck oil stain`、`反绒皮沾水后变黑`
- 风险等级：高

## 适用判断

适用于以下情况：

- 材质表面有明显绒感、方向性、触摸会改变深浅。
- 污渍仍局限在表层或局部范围。
- 还未被大量加水、清洁剂或通用湿巾反复处理。

## 判别要点

1. 白印、浅印、绒毛塌伏，通常更偏向轻度水印或表层摩擦痕。
2. 持续深暗、油亮、边界扩散，通常更偏向油污。
3. 一旦湿擦过重，问题可能从“局部污渍”升级为“绒面整体发硬结块”。

## 处理目标

- 先按污渍类型选工具。
- 尽量维持绒面方向和蓬松度。
- 不把局部污渍扩成更大面积水印。

## 推荐工具

- 翻毛皮专用刷
- 翻毛皮橡皮 / suede eraser
- 若需液体清洁，仅限 suede / nubuck 专用清洁剂，并且必须先测色牢度

## 操作步骤

1. 先完全干燥：任何潮气未退时都不要急着刷洗。
2. 干刷理毛：用专用刷把表层灰尘带走，并观察绒面方向变化。
3. 轻度印痕先用橡皮：对白印、轻微蹭痕、局部倒伏，可先用 suede eraser 做小范围轻擦。
4. 再次刷理：用刷子把绒毛重新立起，判断是否只是表面问题。
5. 只有在需要时才上专用 cleaner：若是局部油污或深色污斑，可在测色后使用 suede / nubuck cleaner，随后吸附、晾干并重新刷理。

## 不建议的做法

- 不要用普通湿巾、酒精、肥皂水或通用皮革清洁剂。
- 不要把清洁液直接大面积喷在皮面上。
- 不要在潮湿状态下猛刷。
- 不要把 smooth leather 的护理油、滋养霜、鞋乳用到翻毛皮上。

## 直接送修 / 升级处理

- 局部持续深黑且有油亮感，提示油污已深入。
- 绒面已经结块、发硬、成片倒伏。
- 清洁后出现明显掉色或色差圈。
- 染色、霉斑、油斑三种风险无法自行明确区分。
