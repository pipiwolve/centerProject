# 常见皮具材质识别与护理差异速查

## 为什么先识别材质

同样是“脏了”“掉色了”“发硬了”，不同材质处理方式可能完全不同。错误识别往往比问题本身更致命，例如把绒面革按光面皮去上油，或把涂层帆布按真皮去深度清洁。

## 常见材质速查

### 植鞣革 / Original Natural Leather

- 常见特征：颜色偏自然、会逐渐氧化变深，常见天然纹理和褶痕。
- 护理特点：怕水、怕强光、怕染色，越天然越要保守处理。

### 小羊皮 / Lambskin / Nappa

- 常见特征：手感细腻柔软，轻压有柔和回弹，表面容易留痕。
- 护理特点：怕热、怕潮、怕摩擦、怕深色衣物和链条压痕。

### Smooth Calf / 光面牛皮

- 常见特征：纹理较细、表面平整，通常有一定光泽和挺度。
- 护理特点：适合温和清洁和基础护理，但不适合强溶剂和高频厚涂。

### Pebble / 颗粒皮

- 常见特征：表面有均匀颗粒纹理，触感更耐看、耐磨。
- 护理特点：相对耐用，但边角和凸起位置仍会因摩擦变浅、失光。

### Suede / 翻毛皮

- 常见特征：绒感明显，手指顺逆方向划过会有颜色深浅变化。
- 护理特点：怕水、怕油、怕重度摩擦，主要靠刷理、吸附和专用喷护。

### Nubuck / 磨砂面革

- 常见特征：也有绒感，但通常比 suede 更厚、更结实，绒面更细密。
- 与 suede 的区别：nubuck 是打磨皮面外层，suede 更接近皮革内层，通常更软更薄。

### Patent Leather / 漆皮

- 常见特征：表面高光、镜面感明显，触感偏滑。
- 护理特点：怕高温、怕光照老化、怕深色染色，也不适合普通光面皮护理油。

### Coated Canvas / 涂层帆布

- 常见特征：重量轻、图案清晰、外层更像带保护层的织物而不是真皮。
- 护理特点：可做轻度湿擦，但要避开酒精、溶剂和浸水。

### Nylon / 尼龙

- 常见特征：轻、柔、实用，常见于通勤、旅行和折叠包。
- 护理特点：总体好清理，但角位、拉链边和提手连接处会先磨损。

## 快速判断方法

1. 看表面：是自然皮纹、规则压纹、绒面还是高亮漆面。
2. 看边缘：真皮切边常能看出纤维层次，涂层帆布和尼龙则更偏织物结构。
3. 看触感：小羊皮最柔，颗粒皮更稳，绒面有明显方向性。
4. 看受损方式：漆皮常见串色和发花，绒面常见结块和油斑，植鞣革常见水痕和氧化变色。

## 护理选择的核心原则

- 不确定材质时，先用最保守的方法：干布、软布、局部测试。
- 绒面、漆皮、浅色天然皮都不适合贸然使用“通用家庭清洁剂”。
- 真正做决定前，先确认它是“光面真皮 / 绒面真皮 / 涂层帆布 / 织物尼龙”中的哪一类。
