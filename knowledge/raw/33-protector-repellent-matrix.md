# 防护喷雾与防水剂矩阵：哪些材质可以考虑，哪些材质不应默认上

## 资料定位

这份矩阵专门解决一个高频误区：用户把“防护喷雾”当成通用安全方案。对云 RAG 来说，关键不是推荐喷雾，而是判断“能不能喷、喷到哪里、喷后要不要停”。

## 总规则

- 任何 protector / repellent 都要先做 hidden spot test。
- 只能在物件干净、干燥、材质已确认的前提下使用。
- 不能因为产品写着“leather”就默认适用于 unfinished natural leather。

## 材质矩阵

### Suede / Nubuck

- 可考虑：suede / nubuck protector
- 适用目标：预防水、油、灰尘
- 前提：先刷净、先测色、薄喷、完全干燥后再使用
- 依据：Coach suede protector；Bickmore Gard-More；Collonil Nubuk + Velours

### Smooth Finished Leather

- 可考虑：在品牌或产品说明明确适用时，使用轻量防护型产品
- 前提：先确认不是 natural vachetta，不是 patent，不是 distressed / napped leather
- 风险：过量喷涂会留下膜感或影响表面手感

### Unfinished / Rawhide / Natural Leather / 本色植鞣革

- 默认不进入通用防水喷雾路线
- 明确限制：Bickmore Gard-More 官方直接写明不用于 unfinished 或 rawhide leather
- 实务意义：对 vachetta / natural cowhide 一类浅色本色皮，云 RAG 应优先给出“谨慎或不建议”的结论

### Patent Leather

- 默认不进入通用喷雾路线
- 原因：高光 finish 需要走 patent-specific care，通用 protector 不是默认安全解

### Canvas / Fabric with Leather Trim

- 可考虑：仅对 fabric/canvas 区域使用 label 明确支持的 protector
- 风险：喷雾漂移到皮边、手柄或滚边后，可能引起色差或手感变化

## 喷涂位阶

1. 先确认材质和 finish。
2. 先清洁并完全干燥。
3. 先在隐蔽处测试。
4. 再做薄层、均匀、非饱和式喷涂。
5. 完全干燥后再继续使用。

## 不建议的做法

- 不要把防护喷雾当作污渍修复方案。
- 不要在湿包、脏包、未测色牢度的包上直接喷。
- 不要在未处理本色皮上盲喷“全材质防水”产品。
- 不要把防护喷雾与多种护理剂连环叠加。
