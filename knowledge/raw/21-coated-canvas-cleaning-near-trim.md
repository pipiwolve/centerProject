# 涂层帆布 / 老花帆布表面脏污：靠近皮边与包耳的安全清洁

## 资料定位

这是一张针对 `coated canvas / monogram canvas / damier canvas / jacquard / canvas with leather trim` 的场景卡，重点解决“帆布能擦、但边上有皮边和走线”的高频难点。

## 检索标签

- 材质：涂层帆布、老花帆布、Monogram canvas、Damier canvas、canvas、jacquard
- 症状：表面脏、白印、手摸发灰、边角轻污、靠近皮边不敢擦
- 典型问法：`老花帆布脏了怎么擦`、`coated canvas cleaning`、`帆布旁边有皮边怎么清洁`
- 风险等级：中

## 适用判断

适用于以下情况：

- 污渍停留在帆布表层，不是渗入性染料或油漆。
- 包体带有皮边、包耳、滚边或走线，清洁时需要避开这些位置。
- 目标是日常表面清洁，不是翻新。

不适用于以下情况：

- 帆布已经发黏、剥膜、开裂。
- 印花区已经磨白。
- 污渍来源为颜料、染料、墨水等高迁移污染。

## 判别要点

1. 帆布可清洁，不代表整只包都可以一起湿擦。
2. 真正高风险的位置通常不是帆布本体，而是皮边、胶边、走线和接缝。
3. 如果污渍靠得很近，清洁策略应以“限制液体扩散”为先。

## 处理目标

- 处理帆布表层脏污。
- 避免让水和清洁剂串到邻近皮革与走线。

## 推荐工具

- 软布
- 少量清水
- 极少量温和皂液或中性洗涤剂，仅限帆布区域
- 干布

## 操作步骤

1. 先除尘：用干布带走表面灰尘。
2. 分区处理：先确认哪些位置是帆布，哪些位置是皮边、包耳、滚边。
3. 轻湿软布：对普通表层脏污，先用微湿软布处理帆布区域。
4. 顽固脏污再加一点点温和皂液：只让布略带清洁力，不让液体自由流动。
5. 立即回收残留：再用干净湿布带走皂液痕迹，随后立刻用干布吸干。
6. 重点看边界：帆布和皮边交界处若有残液，应立即吸走。

## 不建议的做法

- 不要用酒精、手消毒液、香水、皮革油、家庭偏方。
- 不要把清洁液直接喷在包上。
- 不要让液体长时间停留在线边、胶边和皮边。
- 不要为了快而大面积来回刷洗印花区。

## 直接送修 / 升级处理

- 印花或涂层已明显磨损、发白、掉层。
- 接缝处已经伴随皮边、边油或胶边问题。
- 污渍来自强染料、墨水、油漆。
- 清洁后帆布正常，但邻近皮边开始变深、变硬或染色。
