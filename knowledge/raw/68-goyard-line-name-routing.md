# Goyard 系列名分流卡：Saint Louis、Artois、Anjou、Belvedere、Sénat

## 资料定位

这是一张处理 `只说 Goyard 产品线名` 的分流卡。Goyard 的系列名往往同时携带材质结构、使用方式和高频损伤点，适合先做品牌内路由。

## 判别要点

1. Goyard 官方护理边界极保守，line name 一旦识别成功，应优先拉起品牌规则。
2. 同属 Goyardine 家族，不同线的结构差异也很大，不能只按“涂层帆布包”统一回答。
3. 若 query 已明确症状，如 `角磨损`、`拉链问题`、`包口变形`，需在 line routing 后继续进入场景卡。

## 系列路由

### Saint Louis

- 官方材质：Goyardine Canvas & Chevroches Calfskin
- 结构特征：open-top soft tote
- 高频路由：
  - coated canvas + leather trim
  - overload / shape management
  - handle / edge wear

### Artois

- 官方材质：Goyardine Canvas & Chevroches Calfskin
- 结构特征：structured and secure version，带 leather corners 与 zip closure
- 高频路由：
  - corner wear
  - zipper symptom routing
  - leather corner panel stress

### Anjou

- 官方材质：Chevroches Calfskin & Goyardine Canvas
- 结构特征：leather version lined with Goyardine canvas，reversible
- 高频路由：
  - leather-dominant composite
  - reversible surface management
  - handle and fold-line care

### Belvedere

- 官方材质：Goyardine Canvas & Decize Taurillon Leather
- 结构特征：crossbody strap + accessible secure closure
- 高频路由：
  - strap / closure routing
  - coated canvas + taurillon trim
  - front pocket and flap-area wear

### Sénat

- 官方材质：Goyardine Canvas & Chevroches Calfskin
- 结构特征：zip closure + inside card pocket
- 高频路由：
  - zipper routing
  - flat pouch edge wear
  - card-slot / pocket mouth deformation

## 生产端建议

- 用户只说 `Saint Louis` 或 `Artois` 时，先召回本卡和 `67-goyard-material-and-care-map.md`。
- 只要问题牵涉清洁剂选择，优先回忆 Goyard 官方 `avoid commercially available products` 规则。
- 一旦用户目标从“止损”变成“恢复品牌原样”，优先送修或联系 Comptoir。
